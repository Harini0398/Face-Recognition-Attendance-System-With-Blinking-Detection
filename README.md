# Face Recognition Attendance System with Blinking Detection

## Overview
This project is a Face Recognition Attendance System that automates attendance tracking using facial recognition and eye-blink detection for liveness verification.

## Features
- Face Detection
- Face Recognition
- Eye Blink Detection
- Automated Attendance Recording
- Real-Time Processing
- Anti-Spoofing Security

## Technologies Used
- Python
- OpenCV
- NumPy
- Face Recognition Library

## Purpose
To provide a secure and efficient attendance management system that prevents spoofing attacks using photographs.
